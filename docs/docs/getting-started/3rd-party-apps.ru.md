# Сторонние приложения

Мы приветствуем и активно сотрудничаем с разработчиками сторонних прокси-приложений для интеграции поддержки Hysteria в их проекты. Ниже приведён список проектов, поддерживающих протокол Hysteria 2:

> Обратите внимание, что Hysteria 2 — довольно новый проект. Хотя многие проекты поддерживают Hysteria, многие из них имеют в виду старую версию 1.x. Протоколы Hysteria 1.x и Hysteria 2 несовместимы, поэтому убедитесь, что вы используете клиент, специально совместимый с Hysteria 2, при подключении к серверу Hysteria 2.
